# ⏱ Чекпоинт-Таймер для BeamNG.drive

Модификация добавляет в игру **App UI** — функциональную панель на экране, с помощью которой можно записывать и проходить тренировочные или гоночные маршруты.

## 🚀 Основные возможности
1. **Запись маршрутов (Record Route)** — создание стартовой точки, любого количества промежуточных чекпоинтов и финиша. Координаты автоматически сохраняются в файл настроек для текущей карты.
2. **Прохождение маршрута (Race Control)** — таймер автоматически засекает время от старта, фиксирует дельту (разницу) между точками и рассчитывает общее время заезда.
3. **Отображение на миникарте** — точки старта, чекпоинтов и финиша выводятся в виде цветных маркеров на игровой миникарте и карте мира.
4. **История заездов (History)** — сохранение результатов заездов со сплитами по каждой точке для отслеживания прогресса.

## 📦 Установка

### Способ 1 — распакованный (удобно для разработки)
1. Скопируйте папку мода в папку распакованных модов:
   `C:\Users\<Имя_Пользователя>\AppData\Local\BeamNG.drive\current\mods\unpacked\checkpointTimer\`
2. В игре откройте **Vehicle Setup** (иконка гаечного ключа справа) → вкладка **Apps** → найдите **«Checkpoint Timer»** → перетащите на экран.
3. Перезагрузите Lua-скрипты: **Ctrl + L** (движок) и **F5** (интерфейс).

### Способ 2 — запакованный (готовый мод)
1. Положите архив `CheckpointTimer-v3.1.zip` в папку `mods`:
   `C:\Users\<Имя_Пользователя>\AppData\Local\BeamNG.drive\current\mods\`
2. В игре добавьте приложение через меню **Vehicle Setup → Apps**.

## 🎮 Как пользоваться (пошагово)

### Шаг 1. Начать запись маршрута
1. Введите название маршрута в текстовое поле (под заголовком **📍 Record Route**). Если оставить пустым, имя сгенерируется автоматически.
2. Подъедьте к точке старта и нажмите кнопку **New** — старт будет записан.
3. Доезжайте до промежуточных точек и нажимайте кнопку **Add Point** для сохранения каждого чекпоинта.
4. На конечной точке нажмите кнопку **Finish** для сохранения всего маршрута.
   * *Если хотите прервать запись без сохранения — нажмите кнопку **Cancel**.*

### Шаг 2. Выбор и запуск заезда
1. В списке **Routes on Map** кликните на сохраненный маршрут. На игровой миникарте появятся маркеры:
   * 🟢 **Зеленый** — точка старта
   * 🟡 **Желтые** — чекпоинты (ЧК1, ЧК2, ...)
   * 🔴 **Красный** — точка финиша
2. Для начала заезда вы можете:
   * **Авто-режим**: Просто проедьте через стартовую зону. Таймер запустится автоматически.
   * **Ручной режим**: Нажмите кнопку **Start** в блоке **Race Control**.
3. При прохождении чекпоинтов время фиксируется автоматически при въезде в радиус триггера (настраивается в параметрах). Вы также можете нажимать кнопку **CP** вручную.
4. На финише заезд завершится автоматически, либо нажмите кнопку **Finish**. Для сброса заезда используйте кнопку **Reset**.

## ⚙ Настройки (Configuration)

Раздел **Configuration** позволяет настраивать работу приложения:

* **Radius (m)** — радиус срабатывания триггера чекпоинта в метрах. По умолчанию 3.0 м. Если маркеры на миникарте проходятся слишком тяжело или легко, отрегулируйте это значение.
* **Rate (Hz)** — частота опроса позиции машины Lua-скриптом (раз в секунду). По умолчанию 20 Гц.
* **Map: ON/OFF** — включить или отключить отображение цветных маркеров чекпоинтов на игровой миникарте и карте мира.
* **SysClock: ON/OFF** — переключатель источника времени.
  * **ON (Рекомендуется)** — использует системные часы компьютера (`os.clock()`). Защищает от рассинхронизации таймера круга при просадках FPS или лагах симуляции.
  * **OFF** — использует стандартный симуляторный таймер движка.
* **Reload** — перезагрузить файлы сохраненных маршрутов с диска.

## 💻 Консольные команды

Вызываются из консоли игры (`~` или `F11 -> Console`):
`checkpointTimer.ct_command("<команда>")`

| Команда | Описание |
|---|---|
| `help` | Показать справку в консоли |
| `rec start <имя>` | Начать запись нового маршрута |
| `rec next` | Добавить промежуточный чекпоинт |
| `rec finish` | Завершить запись и сохранить маршрут |
| `rec cancel` | Отменить текущую запись |
| `sel <имя>` | Выбрать активный маршрут |
| `del <имя>` | Удалить маршрут |
| `list` | Показать список маршрутов в консоли |
| `start` | Ручной старт заезда |
| `cp` | Ручной проход чекпоинта |
| `finish` | Ручной финиш |
| `reset` | Сбросить текущий заезд |
| `clearhistory` | Очистить историю результатов |
| `sysclock` | Переключить режим системных часов |
| `mapmarkers` | Переключить маркеры на карте |
| `reload` | Перезагрузить конфигурации |
